Capítulo 3 Conjuntos numéricos

Figura 3.1: Número Irracional [Imagen tomada de (zill2012algebra?) pág \(50\)]

Definición3.1 El conjunto de los números naturales consta de:

\[ N=\{ 1,2,3,4,...\} \]

Definición3.2 El conjunto de los números enteros consta de:

\[ Z=\{...,-3,-2,-1,0,1,2,3,4,...\} \]

Definición3.3 El conjunto de los números racionales consta de todos los números que son cociente de dos enteros, siempre que el denominador sea diferente de cero. Es decir:

\[ Q=\{ \dfrac{p}{q} | p \ \text{y} \ q \ \ \text{son números enteros,} \ \ q \ \neq \ 0\} \] ::: {.definition #unnamed-chunk-17} El conjunto de los números irracionales consta de todos los números que no son el cociente de dos enteros, siempre que el denominador sea diferente de cero. Es decir: :::

\[ Q^{*}=\{x | \ x \neq \ \dfrac{p}{q}, \ \ q \ \neq \ 0\ \} \] ::: {.definition #unnamed-chunk-18} El conjunto de los números reales consta de la unión entre el conjunto de los racionales y los irracionales. Es decir: :::

\[ R=\{x | \ x \in Q \ \text{o} \ x \in Q^{*} \}=Q \cup Q^{*} \]
